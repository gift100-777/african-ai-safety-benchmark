# African Multilingual Health AI Safety Benchmark (Pilot)

This project builds a small open benchmark to evaluate hallucinations and safety in AI-generated medical responses across African languages.

## 🎯 Objective
To measure how safe and accurate AI systems are when answering health-related questions in underrepresented languages such as isiZulu, Sesotho, and Setswana.

## 📊 What this project includes
- Multilingual health Q&A dataset (English + African languages)
- AI safety evaluation logic (hallucination detection prototype)
- Simple Streamlit demo for testing responses

## 📁 Current Progress
- Created initial dataset with sample health questions
- Built basic AI safety evaluation script
- Developed a working Streamlit demo interface

## 🧠 Why this matters
AI systems are increasingly used for health information, but they often produce unsafe or incorrect answers, especially in low-resource languages. This project aims to create tools to evaluate and improve that safety.
